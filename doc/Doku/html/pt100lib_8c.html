__s32 pt100_R2T ( __u32  u32Widerstand)

Wandelt einen PT100-Widerstandswert in eine Temperatur um.

Parameter:
u32WiderstandWiderstand in mOhm
Rückgabe:
Temperatur in °mC

Temperatur berechnen

(T2 -T1) x (R - R1)
T = T1 + -------------------
(R2 - R1)
